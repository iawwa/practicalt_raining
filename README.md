# 前端VUE部分

### 如何提交更改？

先提交，然后推送，就可以更改仓库。

### 注意

不要上传node_modules文件夹

## 项目开始

```sh
npm install
```

### 项目运行

```sh
npm run dev
```

### 可能用到,无法使用请使用cnpm替代npm

```
npm install element-plus
npm install axios
npm install vue-router
```

### 如何创建网页

在src//components下创建相应vue文件.再到相应网页使用import使用。

MainBody为页面主体

assets用于存放资源
